## 실습

### 1단계: 프로그램을 실행할 때 접속할 서버의 주소를 입력 받는다.

- `com.eomcs.pms.ClientApp` 변경

### 2단계: 통신 프로토콜 변경에 맞춰 서버의 입력 요청을 처리한다.

프로토콜 예) 
```
[클라이언트]              [서버]
명령         -----------> 작업 수행
예) /board/detail (CRLF)
    CRLF
출력         <----------- 출력 문자열
                          예) 번호?
사용자 입력  <----------- 입력을 요구하는 명령
                          예) !{}!
입력값       -----------> 작업 수행
예) 1
출력         <----------- 출력 문자열
                          예) 제목: 하하하
출력         <----------- 출력 문자열
                          예) 내용: ㅋㅋㅋ
출력         <----------- 출력 문자열
                          예) 작성자: 홍길동
완료         <----------- 빈 문자열
                          예) CRLF
```

- `com.eomcs.pms.ClientApp` 변경
